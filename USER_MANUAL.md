## Термины и определения

### Основные понятия
- **Продукт** - товар, представленный в системе
- **Код продукта** - уникальный идентификатор продукта
- **Дата поступления** - дата закупки товара
- **Срок хранения** - количество дней хранения продукта
- **Объем закупки** - количество закупленного товара (в кг)
- **Объем продажи** - количество проданного товара (в кг)
- **Цена** - стоимость единицы товара (в рублях)

### Аналитические термины
- **Доход** - разница между выручкой от продаж и затратами на закупку
- **Тренд** - динамика изменения показателей во времени
- **Прогноз** - предсказание будущих показателей на основе исторических данных

## Начало работы

### Системные требования
1. **Операционная система:**
   - Windows 10/11 (64-bit)

2. **Минимальные требования к оборудованию:**
   - Свободное место на диске: 200 МБ
   - Разрешение экрана: минимум 1024x768

### Установка программы

1. Перейдите на страницу релизов программы
2. Скачайте версию для вашей операционной системы:
   - Для Windows: `store-windows.exe`
3. Запустите скачанный файл:
   - Windows: Двойной клик по файлу `store-windows.exe`

## Интерфейс программы

### Главное окно

#### Общие элементы
- **Заголовок окна:** "Продуктовый магазин"
- **Размер окна:** минимум 1000x700 пикселей
- **Тема:** темная (для комфортной работы)

#### Вкладки
1. **Вкладка "Управление продуктами"**
   - Управление товарами
   - Добавление/редактирование/удаление продуктов
   - Поиск товаров

2. **Вкладка "Аналитика"**
   - Графики и диаграммы
   - Анализ продаж
   - Прогнозирование доходов

### Вкладка "Управление продуктами"

#### Форма ввода
1. **Поле "Название"**
   - Обязательное поле
   - Пример: "Молоко 2.5%"

2. **Поле "Упаковка"**
   - Тип упаковки товара
   - Пример: "Пластиковая бутылка 1л"

3. **Поле "Дата поступления"**
   - Формат: ДД.ММ.ГГГГ
   - Кнопка календаря справа
   - По умолчанию текущая дата

4. **Поле "Срок хранения"**
   - Количество дней хранения
   - Пример: "14"

5. **Поле "Объем закупки"**
   - Количество закупленного товара (кг)
   - Пример: "100.5"

6. **Поле "Объем продажи"**
   - Количество проданного товара (кг)
   - Пример: "75.0"

7. **Поле "Цена"**
   - Стоимость единицы товара (руб)
   - Пример: "85.50"

#### Кнопки управления
1. **Кнопка "Добавить"**
   - Добавляет новый продукт
   - Активна только при заполнении всех обязательных полей

2. **Кнопка "Редактировать"**
   - Сохраняет изменения в выбранном продукте
   - Активна только при выборе продукта

3. **Кнопка "Удалить"**
   - Удаляет выбранный продукт
   - Требует подтверждения

4. **Кнопка "Поиск"**
   - Поиск продукта по названию

### Вкладка "Аналитика"

#### Графики
1. **График "Тренд дохода"**
   - Динамика изменения дохода по месяцам
   - Прогноз на 3 месяца вперед

2. **График "Топ продаж"**
   - Топ 5 самых продаваемых продуктов
   - Объем продаж в кг

3. **Диаграмма "Распределение запасов"**
   - Доля каждого продукта в общих запасов
   - Круговой график с процентами

## Пошаговые инструкции

### Добавление нового продукта
1. Перейдите на вкладку "Управление продуктами"
2. Заполните все обязательные поля
3. Нажмите кнопку "Добавить"
4. Убедитесь, что продукт появился в таблице

### Редактирование продукта
1. Выберите продукт в таблице
2. Внесите необходимые изменения
3. Нажмите кнопку "Редактировать"
4. Убедитесь, что изменения сохранились

### Удаление продукта
1. Выберите продукт в таблице
2. Нажмите кнопку "Удалить"
3. Подтвердите удаление
4. Убедитесь, что продукт удален из таблицы

### Работа с аналитикой
1. Перейдите на вкладку "Аналитика"
2. Изучите графики и диаграммы
3. Используйте подсказки при наведении на элементы графиков
4. Анализируйте тренды и прогнозы

## Часто задаваемые вопросы

### Общие вопросы
1. **Как добавить новый продукт?**
   - Перейдите на вкладку "Управление продуктами"
   - Заполните форму и нажмите "Добавить"

2. **Почему не сохраняются изменения?**
   Проверьте:
   - Заполнение всех обязательных полей
   - Корректность введенных данных
   - Нажатие кнопки "Редактировать"

3. **Как обновить графики?**
   - Графики обновляются автоматически
   - При необходимости перейдите на вкладку "Аналитика"

### Работа с данными
1. **Можно ли восстановить удаленный продукт?**
   - Нет, удаление необратимо
   - Всегда проверяйте перед удалением

2. **Почему не строится прогноз?**
   Необходимо:
   - Минимум 3 месяца данных
   - Регулярные данные о продажах

3. **Как экспортировать данные?**
   - В текущей версии экспорт не реализован
   - Используйте скриншоты графиков

## Устранение проблем

### Проблемы с вводом данных
1. **Не принимается название продукта**
   - Проверьте уникальность
   - Используйте допустимые символы

2. **Ошибка в дате**
   - Используйте формат ДД.ММ.ГГГГ
   - Используйте календарь

3. **Не сохраняются изменения**
   - Проверьте заполнение всех полей
   - Проверьте формат данных
   - Нажмите правильную кнопку

### Проблемы с аналитикой
1. **Не обновляются графики**
   - Проверьте наличие данных
   - Перезапустите программу

2. **Некорректный прогноз**
   - Проверьте количество данных
   - Убедитесь в регулярности данных

3. **Пустые графики**
   - Проверьте наличие данных
   - Проверьте выбранный период

### Общие проблемы
1. **Программа работает медленно**
   - Закройте лишние программы
   - Проверьте свободное место
   - Очистите временные файлы

2. **Окно не масштабируется**
   - Проверьте разрешение экрана
   - Минимальный размер 1000x700

3. **Проблемы с отображением**
   - Обновите драйверы видеокарты
   - Проверьте разрешение экрана
   - Перезапустите программу

   - Анализируйте продажи
   - Выявляйте популярные товары
   - Предлагайте улучшения

1. **Регулярное сохранение**
   - Создавайте резервные копии базы данных

2. **Контроль доступа**
   - Ограничьте доступ к программе

3. **Проверка данных**
   - Проверяйте вводимые данные
   - Исправляйте ошибки сразу
